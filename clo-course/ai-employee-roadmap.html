"When do I…"

The quick gut-check for which tool fits. (Claude makes these calls with you — this is just so the words make sense as you go.)

…split a skill?
When it's trying to do two jobs and keeps forgetting one. Two jobs → two skills that hand off to each other.
…add a harness?
When a skill works but drifts, or breaks its own rule. A harness is an automatic check that catches it every time — stronger than adding more instructions.
…use an agent?
When the work should run in the background or in parallel — "go do this and report back," not walked through with you.
…make a command?
When you do the same thing the same way, often, and want a one-word shortcut for it.
…wire a routine?
When it should run on a schedule — nightly, weekly — or fire on an event, with no one starting it.
…just build the next one?
When the current employee is solid on all of the above. Map the next bottleneck and repeat the ladder.

Starter prompts

Don't start from a blank page. Pick your kind of work, copy a prompt, paste it into Claude Code, and it builds the skill with you. Tweak the wording to fit your business.

Build me a skill that turns a YouTube transcript into a first-draft newsletter in my voice, with 3 subject-line options.

Build me a skill that repurposes one newsletter into a Twitter/X thread and a LinkedIn post, keeping my voice.

Build me a skill that takes a rough idea and returns 5 hooks, with the angle behind each one.

After it's built, say "harden this skill" to add a voice check so the drafts stay on-brand.

Build me a skill that turns my sales-call notes into a proposal with scope, timeline, and 3 pricing options.

Build me a skill that drafts a follow-up email sequence after a sales call, using the prospect's stated goals.

Build me a skill that preps me for a sales call from the prospect's website and our past emails.

Running this for every lead? Say "make a command for this" so it's one word.

Build me a skill that turns a signed client into an onboarding checklist, a welcome email, and an intake form.

Build me a skill that writes a weekly client status update from my project notes.

Build me a skill that turns a finished project into a case-study draft and a testimonial request.

Onboarding has several steps? Say "build a skill system for this" to chain them into one flow.

Build me a skill that turns my raw numbers into a monthly business report with the 3 things that moved.

Build me a skill that triages my inbox into reply-now / delegate / archive, with draft replies.

Build me a skill that turns a messy meeting transcript into decisions, action items with owners, and open questions.

Want the report every month on its own? Say "run this on a schedule".

Back in Claude Code, say "what's next" anytime to pick up where you left off.
