FAQ
Do I need to know how to code?
No. You copy a few short commands into a chat box and click "Install." If you can copy and paste, you can do this — and the "Let Claude install it" prompt up top will walk you through it.
Why install SkillStack first, then clo-community?
SkillStack is the secure delivery system for paid plugins — like the app store. You add it once; it checks your license and hands you the exact commands to install clo-community. Two installs, one time.
Does my license key work on more than one computer?
Yes. The same key activates on every computer you use — run /activate-license and paste it on each one. No new key per device.
Why does it need Node.js?
SkillStack delivers the plugin as a secure npm package, and Node.js (which includes npm) downloads it. Install once from nodejs.org and never touch it again.
How do updates work?
On a subscription, every update ships automatically — especially with auto-update on (Step 5 tip). To apply updates manually any time, run /skillstack:update-plugins.
What if my subscription lapses?
Access pauses until you renew. Renew at Lemon Squeezy → My Orders, then run /activate-license again.
I already use Claude Code — can I skip ahead?
Yes. If the extension and Node.js are installed, tick Step 1 and jump to Step 3.
Is the Terminal method different from VS Code?
Same result, same commands — just a different place to type them. The VS Code way is click-based and friendly; Terminal is for command-line folks. Use the toggle at the top to switch.
